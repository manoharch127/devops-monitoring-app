# devops-monitoring-app
Built an end-to-end CloudOps monitoring platform integrating AWS EC2, Docker, Jenkins CI/CD, CloudWatch monitoring, SNS, Lambda, and ServiceNow incident workflows. Implemented automated alerting and incident generation through event-driven architecture.
